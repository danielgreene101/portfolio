Movie Organizing Aplication

Movies and APIs

This project uses a movie API from themoviedb.org, and uses fuzzy search to create a movie app that allowed users to search, save, and rate movies. The group used Bootstrap, HTML/CSS, JavaScript, Jquery, and fuzzy search to reach our MVP. The coding experience of creating this app was unlike prior projects in that because we only had one member coding, while all three of us contributed to the plan. This resulted in very few errors, and understanding each other's thought process very clearly.
