About

How I got where I am today.

  • February - 2016

    Started Coding

    This is when I begin coding with the very basics. Working through Mark Myers, 'A Smarter Way to Learn' books I built a firm coding foundation. This foundation allowed me to gain admissions to attend the Nashville Software School later that year.

  • July - 2017

    Boot Camp

    At this point I went from coding around an hour a day, to well over 8 hours. At Nashville Software School, we began to learn not only the fundamentals of the front end through JavaScript, HTML and CSS, but also how to better learn programming, along with best practices to optimize learning potential.

  • October - 2017

    The Back End

    At this stage in our course, we began to learn backend programming through the languages Ruby and Ruby on Rails. I had completed a front-end capstone to graduate to this point. That website, a dungeons and dragons character creator, is still up and active with a link provided above within my portfolio.

  • January 2018

    Graduation

    This coming January I will be graduating from the NSS program and entering the programming job market. Wish me luck!
