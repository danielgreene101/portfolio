About

The journey to get where I am today.

  • February - 2016

    Started Coding

    This is when I begin coding with the very basics. Working through Mark Myers, 'A Smarter Way to Learn' books, I built a firm coding foundation. This helped me gain admissions to the Nashville Software School

  • July - 2017

    Boot Camp

    Admitted to Nashville Software School (NSS). Coded in excess of 8 hours a day. Learned the fundamentals of front end coding through JavaScript, HTML and CSS, as well as better programming techniques and best practices to optimize learning potential. Completed front-end capstone: Dungeons and Dragons Character Creator

  • October - 2017

    The Back End

    Learned back-end programming languages Ruby and Ruby on Rails.

  • January 2018

    Graduation

    This coming January I will graduate from the NSS program and entering the programming job market. Wish me luck!
